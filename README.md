# myapp

Static HTML site deployed via Cloudflare Pages.

## Local preview
Open `index.html` in your browser.

## Deploy (Cloudflare Pages)
1. Push this repo to GitHub.
2. In Cloudflare Dashboard → **Workers & Pages** → **Pages** → **Create a project** → **Connect to Git**.
3. Framework preset: **None**
4. Build command: *(leave empty)*
5. Build output directory: `/` (root)

`index.html` at repo root is the entry.
